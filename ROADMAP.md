# SELENE Roadmap

This document outlines the planned development direction for SELENE.

## Version 0.2.0 (Q1 2026)

### Features
- [ ] **Export/import functionality** - Backup/restore complete user data
- [ ] **Enhanced visualizations** - Trend graphs for symptoms over time
- [ ] **Notification system** - Reminders for daily attune logging

### Technical Improvements
- [ ] **Performance optimization** - RAG query caching improvements
- [ ] **Model flexibility** - Support for multiple LLM backends (Ollama, LocalAI)
- [ ] **Mobile responsiveness** - Improved UI for mobile browsers

## Version 0.3.0 (Q2 2026)

### Features
- [ ] **Symptom predictions** - ML-based forecasting using historical patterns
- [ ] **Medication tracking** - Log medications and correlate with symptoms
- [ ] **Custom symptom definitions** - User-defined tracking metrics
- [ ] **Journaling mode** - Free-form notes with tagging

## Version 1.0.0 (Q3 2026)

### Features
- [ ] **Clinician sharing** - Secure, anonymized report export for healthcare providers
- [ ] **Community knowledge base** - Opt-in aggregated anonymized insights
- [ ] **Multi-language support** - i18n for global accessibility

### Technical Improvements
- [ ] **Security hardening** - Penetration testing and audit
- [ ] **Accessibility (a11y)** - WCAG 2.1 AA compliance
- [ ] **Performance benchmarks** - Documented performance targets

## Long-term Vision

### Research & Innovation
- Federated learning for privacy-preserving model improvements
- Integration with wearable devices (Fitbit, Apple Health)
- Voice interface for hands-free logging
- Differential privacy for anonymized data sharing

### Community & Ecosystem
- Academic partnerships for menopause research
- Open datasets (anonymized, opt-in) for researchers
- Educational content integration
